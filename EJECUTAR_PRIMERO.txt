PUNTO DE VENTA CELULARES - LINOEM DEVELOPMENT
================================================

PARA SUBIR A GITHUB AUTOMATICAMENTE EN WINDOWS:
1. Descomprime el ZIP completo.
2. Abre la carpeta punto-de-venta-celulares.
3. Haz doble clic en SUBIR_A_GITHUB_AUTOMATICO.bat.
4. El archivo NO utiliza winget. Descarga GitHub CLI portable directamente desde GitHub.
5. Inicia sesion en la ventana oficial de GitHub cuando se abra.
6. El script creara o actualizara:
   https://github.com/emannnuel55-pixel/PUNTO-DE-VENTA-CELULARES

PARA PROBAR LA PLATAFORMA CON DOCKER:
1. Abre Docker Desktop.
2. Haz doble clic en INICIAR_CON_DOCKER.bat.
3. Abre http://localhost:3000
